## Environment Setup

### Mac

1.Install Python3

https://www.python.org/downloads/

2.Install pandas

```
pip3 install pandas
```

3.Install lxml

```
pip3 install lxml
```

4.Install tushare

https://pypi.python.org/pypi/tushare/

```
python setup.py install
```

5.Check

```
import tushare

print(tushare.__version__)
```
